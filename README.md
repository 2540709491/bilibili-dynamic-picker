B站动态 API 跳转助手
本项目由DeepSeekV4生成,参考https://github.com/tongle2025/bilibili-dynamic-jumper
并附带一个黑历史删除工具.
通过 Bilibili 官方 API 快速定位 UP 主在指定日期发布的动态，彻底告别无限滚动。

✨ 功能特点
API 直连：直接调用 B 站动态接口，翻页速度快，无需等待页面加载。

精准定位：支持按年、月、日（可选）查找，自动判断目标时间范围。

详细进度：实时显示已翻页数、找到的动态数、当前翻至的最早动态日期。

结果高亮：找到的动态以列表展示，点击可直接跳转到对应动态页面。

随时中断：支持随时停止搜索，进度保留不丢失。

轻量无依赖：纯前端脚本，无需任何第三方库。

📥 安装方法
在你的浏览器中安装 Tampermonkey 扩展。

点击 Tampermonkey 图标 → “创建新脚本”。

将脚本代码完整复制粘贴进去，替换原有内容。

按 Ctrl+S 保存。

或者直接通过脚本分享链接安装（如有提供）。

🚀 使用说明
打开任意 UP 主的空间动态页面，例如：
https://space.bilibili.com/111/dynamic 或 https://t.bilibili.com/111

页面右上角会出现一个 📅 浮动按钮（或完整控制面板）。

在面板中选择：

目标年份（如 2022 年）

目标月份（如 3 月）

目标日（可选，若不选则查找该月所有动态）

点击 “🚀 通过 API 搜索”，脚本将自动翻页查找。

进度面板会显示：

已翻页数

已找到的动态数量

当前翻至的最早动态日期

目标时间范围

搜索完成后，会弹出结果列表。点击对应链接即可跳转到具体动态。

如需提前终止，点击 “⏹️ 停止搜索” 即可。

⚠️ 注意事项
登录状态：脚本需要你的 B 站登录凭据（Cookie）。确保你已在该浏览器登录 B 站，否则 API 可能返回错误。

搜索耗时：如果 UP 主动态数量庞大（数万条），搜索早期时间可能需要 1~3 分钟，请耐心等待。

频率限制：短时间内大量请求可能触发 B 站接口限制，如遇到异常请稍后再试。

失效处理：动态若已被删除或隐藏，API 将不会返回，因此不在搜索结果中。

匹配逻辑：

若选择了具体日期，只匹配当日动态。

若只选年月，匹配该月全部动态。

动态时间以 PUBLISH 时间戳为准（pub_ts），非编辑时间。

适用范围：目前仅支持 UP 主空间动态页面，不支持首页推荐流。

🔧 脚本版本

🛠 技术原理
脚本通过 fetch 调用 B 站接口：

text
GET https://api.bilibili.com/x/polymer/web-dynamic/v1/feed/all
参数：

host_mid：UP 主 UID

offset：翻页偏移量

platform=web

features：固定参数

对每页返回的动态，检查其 pub_ts 时间戳是否落在目标日中。
当最旧动态的日期早于目标范围时，自动停止搜索，避免无谓请求。

B站评论批量删除工具（油猴脚本）
一个在 B 站个人主页运行的油猴脚本，可根据 aicu 导出的历史评论 JSON 数据，筛选并批量删除自己的评论。
依赖于https://www.aicu.cc/
aicu数据指在该网站查询评论后F12获取到的如图所示的响应json数据

✨ 功能
手动输入 bili_jct 和 SESSDATA，无需脚本自动读取 Cookie（更安全）

粘贴 aicu 导出的完整 JSON 数据，自动解析评论列表

界面清晰显示评论内容、时间、类型、目标 ID

支持全选、反选，可自由勾选需要删除的评论

批量删除选中评论，并显示实时进度和结果

面板可拖拽移动，不影响页面正常浏览

📖 使用方法
安装脚本
在已安装油猴（Tampermonkey）扩展的浏览器中，新建脚本并复制本脚本全部代码，保存即可。

进入 B 站个人空间
脚本仅在 https://space.bilibili.com/* 生效，请先打开自己的个人主页。

填写凭证
在页面右侧出现的面板中，分别输入：

bili_jct（CSRF Token）

SESSDATA

如何获取？ 登录 B 站后，按 F12 打开开发者工具 → 应用程序（Application）→ Cookie → 选择 bilibili.com，在列表中寻找 bili_jct 和 SESSDATA，复制其值。

粘贴 aicu 数据
打开 aicu 导出或获取到的评论 JSON 数据，复制全部文本，粘贴到面板的大输入框中，点击 加载评论。

选择并删除

勾选你要删除的评论（可使用全选/反选按钮）

点击 删除选中，确认后开始逐条删除

进度和结果会实时显示

⚠️ 注意事项
权限说明：本脚本不会收集、上传或存储你的任何个人信息，所有请求直接由你的浏览器发送至 B 站官方 API。

凭证安全：bili_jct 和 SESSDATA 是敏感凭证，请勿分享给他人。使用完毕后，建议清除输入框或关闭页面。

删除不可恢复：评论一旦删除无法找回，请谨慎勾选。

频率控制：脚本已内置 300ms 间隔，避免请求过快触发风控。如仍有频繁提示，请适当减少单次删除数量。

兼容性：仅支持 B 站个人主页 (space.bilibili.com)，其他页面无法启动面板。

数据来源：JSON 数据格式需与 aicu 导出的评论列表一致（包含 code、data.replies 字段）。

🛠️ 常见问题
Q：为什么点击删除提示“未找到凭证”？
A：请确保已正确填写 bili_jct 和 SESSDATA，这两个值与登录账号一一对应。

Q：加载 JSON 后没有显示评论？
A：请确认粘贴的是完整的 aicu 导出 JSON（如 {"code":0,"data":{"replies":[...]}}），而不是只复制了数组。

Q：删除失败，错误码 -101 或 -111？
A：-101 表示登录态失效，请重新登录 B 站并更新 SESSDATA；-111 表示 CSRF 校验失败，请检查 bili_jct 是否正确。

Q：可以删除别人的评论吗？
A：不能。B 站接口只允许删除自己发送的评论，或自己管理的评论区下的评论。
